# RAG-Enhanced LLM-Driven Detection Engineering for SOC Operations

## Overview

This repository contains my master's research project on AI-assisted Detection Engineering for Security Operations Centers (SOCs).

The objective of this research is to investigate how Retrieval-Augmented Generation (RAG) and Large Language Models (LLMs) can assist SOC analysts during alert investigation, contextual threat analysis, and detection engineering activities.

Rather than replacing security analysts, this project explores how AI can augment existing SOC workflows by combining contextual knowledge retrieval, MITRE ATT&CK mapping, threat intelligence, and LLM reasoning.

The project focuses entirely on defensive cybersecurity and aims to develop a practical proof-of-concept using open-source technologies.

---

## Research Objectives

- Investigate the application of LLMs in Security Operations Centers.
- Explore RAG for contextual threat analysis.
- Integrate MITRE ATT&CK knowledge into AI-assisted investigations.
- Support Detection Engineering using AI.
- Evaluate the effectiveness of AI-assisted SOC workflows.
